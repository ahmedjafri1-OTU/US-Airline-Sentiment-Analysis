# Twitter Airline Sentiment Analysis  <img src="https://img.icons8.com/?size=100&id=6Fsj3rv2DCmG&format=png" width="30"/> 
### Project Overview  <img src ="https://img.icons8.com/?size=160&id=105278&format=png" width="30"/>
This project analyzes customer sentiment toward major U.S. airlines based on tweets posted in February 2015. The dataset contains tweets where travelers expressed their experiences and opinions about different airlines on Twitter.
The goal of the project is to explore how passengers feel about airline services and visualize the sentiment distribution across different airlines.
### Dataset <img src="https://img.icons8.com/?size=160&id=118369&format=png" width="30"/>
The dataset contains tweets related to major U.S. airlines. Each tweet is labeled with a sentiment indicating the passenger's opinion about the airline.
Airlines Included

* Virgin America
* US Airways
* United Airlines
* American Airlines
* Delta Airlines

### Sentiment Categories 
The tweets are classified into three sentiment categories:
* Positive <img src = "https://img.icons8.com/?size=96&id=63230&format=png" width= "20"/>
* Neutral  <img src = "https://img.icons8.com/?size=96&id=63224&format=png" width= "20"/>
* Negative  <img src = "https://img.icons8.com/?size=96&id=13770&format=png" width= "20"/>
  
These labels represent how travelers expressed their experiences with airline services on Twitter.

### Analysis Performed <img src="https://img.icons8.com/?size=160&id=42220&format=png" width="30"/>
The project performs several analyses to understand customer sentiment:

* Distribution of tweets across different airlines
* Sentiment distribution (positive, neutral, negative)
* Visualization of tweet activity over time
* Word cloud analysis to identify commonly used words in tweets

### Interactive Map Visualization <img src="https://img.icons8.com/?size=160&id=118630&format=png" width="30"/>
The application includes an interactive map visualization built using Streamlit.
This feature allows users to:

* View tweet locations on a geographic map
* Explore where passengers are posting tweets from
* Observe sentiment patterns based on location
  
The map helps provide spatial insight into where airline-related discussions are occurring and allows users to interactively explore the geographic distribution of tweets.

### Streamlit Application  <img src="https://img.icons8.com/?size=96&id=Rffi8qeb2fK5&format=png" width="30"/>
The interactive dashboard for this project is built using Streamlit.
The application allows users to:

* Explore random tweets by sentiment
* Visualize sentiment distribution across airlines
* View tweet frequency over time
* Generate word clouds to highlight common words used in tweets
  
This dashboard provides an intuitive way to analyze airline sentiment and understand customer opinions.
